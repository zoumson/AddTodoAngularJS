This project use angularJS as framework
The main functionality of this app is to allow user to key in his or her tasks to do, for fear to forget them.
For each task done, user can mark the checkbox in front of that given task.
The checked task is marked as done using red line passing through that task.
User can individually remove each task using the delete button at the right.
For the finished task, user can choose to remove them by pressing the delete done task button.
At the bottom of the website, the number of total task is mentioned.
